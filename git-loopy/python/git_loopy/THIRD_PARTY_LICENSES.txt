Third-Party Licenses
====================

git-loopy vendors and redistributes a workflow skill catalog inside the wheel as
package data, so `git-loopy init` can scaffold it without a source checkout. The
catalog is assembled from third-party, open-source skills; the copyright and
permission notices for every component git-loopy redistributes travel with the
distribution and are reproduced (or referenced) below.

The three optional tool/vendor integrations that git-loopy does NOT vendor
(microsoft-docs, microsoft-foundry, playwright-cli) are cleanly severable and are
not redistributed in the wheel.
