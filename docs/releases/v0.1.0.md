# git-loopy 0.1.0

git-loopy 0.1.0 is the first stable source distribution of the phase-1 Runner
family. It includes the Python reference Orchestrator and native shell and
PowerShell Orchestrators, all implementing the same bounded autonomous loop:
AFK-ready issue collection, one Active issue per Iteration, Runner-owned
Checkpoints and push, progress and Strike accounting, verified issue closure,
and JSONL replay.

## Compatibility

- Every public Orchestrator identity surface reports the exact shared Release
  version `0.1.0`: `git-loopy --version`, `wrapper.run.start`, and the
  Continuation capability manifest.
- Release identity is independent of compatibility identity. This distribution
  implements Wrapper contract 1.3, Event schema fixture 1.1 (compatibility
  schema 1), Continuation contract 1.0, and Continuation record format 1.
- The Runner-family Conformance gate covers the Python, shell, and PowerShell
  production boundaries before publication.

## Installation

- Download GitHub's automatic source archive and run the selected Orchestrator
  from the extracted tree.
- Python 3.11 or newer can install the nested Python project directly from the
  `v0.1.0` tag with `uv tool install
  "git+https://github.com/bradcstevens/git-loopy@v0.1.0#subdirectory=git-loopy/python"`.
- The clone-local shell and PowerShell installers add launchers for their
  native Orchestrators. Shell requires Bash 4+ and `jq`; PowerShell requires
  PowerShell 7+.

This source-only Release does not publish to PyPI, Homebrew, winget, or Scoop,
and does not include signed platform artifacts or a compiled TUI helper.
